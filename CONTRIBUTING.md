# CONTRIBUTING.md

This repository is not open to external contributions.

- Pull requests and forks are not accepted.
- The `main` branch is protected; direct pushes are restricted to the project owner.
- Issue tracking and discussions are disabled.
- Architectural decisions and internal methodology live in files that are intentionally not part of this public repository.

For business inquiries, visit [rotative.com.ar](https://rotative.com.ar) or contact `contacto@rotative.com.ar`.
